## Course Topics (mainly 4 parts)

- Rasterization
- Curves and Meshes
- Ray Tracing
- Animation / Simulation

### Rasterization

- 将几何图元（三维三角形/多边形）投影到屏幕上
- 将投影的图元分解为片段
- 实时应用（如游戏中）

### Curves and Meshes

- 如何在计算机图形学中表示几何

### Ray Tracing

- 从相机中射出的光线穿过每个像素
  - 计算交叉点和阴影
  - 继续反弹光线，直到它们击中光源
- 离线应用（如电影，动画）

### Animation / Simulation

- 关键帧动画
- 弹簧质点系统
